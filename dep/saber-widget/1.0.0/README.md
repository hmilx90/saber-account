saber-widget [![Build Status](https://travis-ci.org/ecomfe/saber-widget.png)](https://travis-ci.org/ecomfe/saber-widget)
===

适合移动端的UI组件库。

## Installation

通过 [`edp`](https://github.com/ecomfe/edp) 引入模块：

```sh
edp import saber-widget
```
